🌍 WanderVista – Travel Website UI Design (Figma Project)
📌 Project Overview

WanderVista is a modern and visually engaging travel website UI designed in Figma.
The goal of this project was to create a clean, premium, and user-friendly interface for a travel booking platform that helps users explore destinations, read traveler stories, and book their next adventure easily.

This project focuses on:

Clean layout structure

Modern UI trends

Attractive hero section

Card-based destination browsing

Strong visual hierarchy

Consistent color system

🎯 Project Objectives

Design a professional travel landing page UI

Apply modern UI/UX principles

Create reusable components

Maintain consistency in typography and spacing

Ensure responsive layout structure (Desktop-first design)

Improve user engagement through visuals and CTA placement
